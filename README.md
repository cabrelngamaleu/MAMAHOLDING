# MAMAHOLDING - Excellence & Innovation

Un site web premium et sophistiqué pour MAMAHOLDING, conçu pour surpasser les standards de l'industrie avec un design moderne et élégant.

## 🌟 Caractéristiques

### Design Premium
- **Palette de couleurs luxueuse** avec des dégradés dorés sophistiqués
- **Typographie élégante** utilisant Playfair Display et Poppins
- **Animations fluides** et transitions professionnelles
- **Effets de parallaxe** pour une expérience immersive
- **Design responsive** optimisé pour tous les appareils

### Sections du Site

1. **Hero Section** - Introduction accrocheuse avec animations de formes dynamiques
2. **Stats Section** - Compteurs animés affichant les réalisations
3. **Services** - Présentation des services avec cards interactives
4. **Portfolio** - Galerie de projets avec overlays élégants
5. **Expertise** - Mise en avant de l'expérience et des compétences
6. **À Propos** - Histoire et valeurs de l'entreprise
7. **Témoignages** - Avis clients avec système d'étoiles
8. **Contact** - Formulaire interactif et informations de contact

### Technologies Utilisées
- HTML5 sémantique
- CSS3 avec variables CSS et Grid/Flexbox
- JavaScript vanilla (ES6+)
- Google Fonts (Playfair Display, Poppins)
- Images haute qualité via Unsplash

### Fonctionnalités JavaScript
- Navigation smooth scroll
- Animations au scroll (Intersection Observer)
- Compteurs animés pour les statistiques
- Menu mobile responsive
- Effets de parallaxe
- Curseur personnalisé (desktop)
- Effets ripple sur les boutons
- Lazy loading des images
- Formulaires interactifs

## 🚀 Installation

1. Clonez le repository :
```bash
git clone https://github.com/votre-username/mamaholding.git
```

2. Ouvrez simplement `index.html` dans votre navigateur préféré

Aucune installation de dépendances n'est nécessaire - le site fonctionne avec du HTML, CSS et JavaScript pur !

## 🎨 Personnalisation

### Couleurs
Les couleurs principales sont définies dans `:root` du fichier `style.css` :
```css
--primary-color: #1a1a2e;
--accent-color: #e94560;
--gold: #d4af37;
```

### Images
Remplacez les URLs Unsplash par vos propres images dans `index.html`

### Contenu
Modifiez le texte directement dans `index.html` pour personnaliser :
- Nom de l'entreprise
- Services offerts
- Témoignages
- Coordonnées

## 📄 Structure des Fichiers
